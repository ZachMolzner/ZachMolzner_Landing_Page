# Zach Molzner Portfolio

## About This Project

This is my personal portfolio site. I built it to show who I am as a software engineer, educator, and instructional designer. The goal is to give employers a clear look at my technical skills, teaching background, projects, and the type of work I want to keep building.

## Why I Built This

I wanted one place where I could connect both sides of my background: software development and education. My experience in the classroom taught me how to explain complex ideas, design learning experiences, support different users, and solve problems under pressure. My software engineering work lets me take that same mindset and build tools, interfaces, and systems that are clear, useful, and practical.

## Featured Work

### Ivalice Chronicles / Final Fantasy Tactics Party Optimizer

Ivalice Chronicles is a tactical RPG party optimizer inspired by Final Fantasy Tactics. I built it to practice React, user-centered interface design, and systems thinking. The project focuses on helping users compare jobs, roles, abilities, and party coverage so they can make stronger team-building decisions.

### SarahNode AI Assistant

SarahNode is a local-first AI assistant project that explores how a personal AI companion can feel useful, responsive, and practical. The work includes React, FastAPI, WebSockets, voice features, avatar interaction ideas, and event-driven architecture.

### Curriculum and Instructional Design Materials

My education background is still a major part of how I work. I include curriculum and instructional design materials to show how I plan learning experiences, write clear directions, build assessments, scaffold complex ideas, and create resources that support different types of learners.

## Running the Project Locally

Install the project dependencies:

```bash
npm install
```

Start the local development server:

```bash
npm run dev
```

Build the production version:

```bash
npm run build
```
